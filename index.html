PAT-1025-反转链表

发表于 2020-02-12 | 分类于 PAT

静态链表，通过数组模拟，数组下标表示地址。
结点定义 需要加一个标记 因为可能给出不连在链表里的结点。
注意最后不足K个不反转。

1.可通过按标记排序把所有结点放到前面,控制输出完成逆序

struct Node {
    int adress, data, next;
    int order;  //结点在链表上的序号 无效记为maxn 
} node[maxn];

2.也可以模拟链表真实进行反转，

typedef struct Node{
    int address;
    int num;
    int next;
    struct Node *link;
}LNode, *Pin;
