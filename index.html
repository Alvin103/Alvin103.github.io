解决网页图片加载失败

  • 图片问题
  • 解决方法

图片问题

浏览某网页时发现头像等图片都挂掉了，刚开始以为是加载不出来，后来在浏览vscode插件的介绍中发现也不能加载图片。
github       vscode （忘记截图了^_^）

解决方法

经过搜索，原因可能是DNS污染，通过修改hosts文件解决了。

  • C:\Windows\System32\drivers\etc 目录下 hosts 添加内容

网址可通过浏览器检查 找到
然后根据src查询域名
